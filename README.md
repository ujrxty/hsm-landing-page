# Home Services Mastery

A high-converting landing page for a home services coaching business built with React, Framer Motion, and Tailwind CSS.

## Features

- Modern, responsive design with dark theme
- Hero section with embedded YouTube VSL
- Results/stats section with animations
- Testimonials carousel with 40+ images
- Multi-step typeform-style application form
- Admin dashboard to view applications
- Email notifications for new applications
- Calendly integration for booking calls

## Quick Start

1. Install dependencies:
\`\`\`bash
npm run setup
\`\`\`

2. Start development servers:
\`\`\`bash
npm run dev
\`\`\`

Frontend: http://localhost:5173
Backend: http://localhost:3001

## Configuration
