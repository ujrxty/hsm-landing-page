# Home Services Mastery Landing Page

## Project Overview
Landing page for Home Services Mastery coaching business. Dark theme, simple layout replicating homeserviceascension.com/vsl62893763.

## Tech Stack
- React + Vite
- Tailwind CSS
- Deployed on Render (auto-deploys from GitHub)
- GitHub repo: https://github.com/ujrxty/hsm-landing-page

## Development
```bash
npm run dev          # Start dev server on localhost:5173
npm run build        # Build for production
```

## Structure
- `src/pages/LandingPage.jsx` - Main landing page component
- `public/testimonials2/` - Testimonial images (1.jpeg through 12.jpeg)

## Key Components
- Hero section with main YouTube video (ID: z28HxuI9X-4)
